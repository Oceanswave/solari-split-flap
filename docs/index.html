Solari Split-Flap Display

A faithful recreation of the classic split-flap departure board, built as a React + TypeScript component.

React 17+ TypeScript Zero Dependencies 3.2 kB gzip

Get Started

npm install solari-split-flap
import { SolariBoard } from 'solari-split-flap';

function App() {
  return <SolariBoard sound />;
}

Features

Authentic Animation

CSS 3D transforms with perspective and backface-visibility. Characters drum through in sequence with decelerating easing.

Synthesized Sound

Web Audio API click on every flap — white noise burst through a highpass filter with exponential gain decay.

Per-Line Colors

Any row can have its own color. Use the @ prefix shorthand for gold author attribution lines.

Controlled Mode

Use the value prop for standard React controlled behavior. Board animates whenever value changes.

Imperative Ref

Call ref.flipTo() and ref.clear() directly for programmatic control outside React state.

Responsive Scale

A single scale prop proportionally resizes cells, gaps, padding, and font size for any viewport.

Custom Drum

Supply your own character sequence via the drum prop. Characters cycle through your drum on every flip.

Zero Dependencies

Only peer-depends on React. No runtime dependencies. Tree-shakeable ESM with full TypeScript declarations.
